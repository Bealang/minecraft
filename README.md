# Minecraft Chat Formatter

### The Ultimate Minecraft Text Formatter

Minecraft Chat Formatter is a lightweight, web-based utility designed for server administrators, developers, and players who want to master Minecraft’s text formatting system. Built with a retro GUI aesthetic that mirrors the authentic game experience, it allows you to create complex, colorful, and styled strings for chat, signs, MOTDs, and plugin configurations.

---

## ✨ Key Features

* **Legacy & Format Support:** Full support for standard color codes (0-f) and formatting styles (Bold, Italic, Underline, Strikethrough).
* **Live Interactive Preview:** See exactly how your text will look in-game as you type, including a functional **"Magic" (Obfuscated)** effect animation.
* **Special Character Library:** Quick-access buttons for frequently used symbols such as `⚠`, `»`, `┃`, `❤`, and `★` to make your messages stand out.
* **Local Preset System:** Save your favorite templates directly in your browser. You can name, rename, overwrite, or delete presets without needing an external database.
* **One-Click Copy:** Instantly copy your formatted string to the clipboard, ready to be pasted into Minecraft or your server config files.
* **Authentic Design:** A fully responsive interface featuring pixel-accurate Minecraft textures, fonts, and button behaviors.

---

## 🚀 How to Use

1. **Input:** Type your message in the main text area.
2. **Style:** Click the color grid or formatting buttons to insert codes (e.g., `&a` for green, `&l` for bold) at your cursor position.
3. **Symbols:** Enhance your text by clicking any of the extra symbols in the toolbar.
4. **Preview:** Check the "Live Chat Preview" section to ensure the colors and styles are perfect.
5. **Save/Copy:** * Give your creation a name and hit **SAVE** to store it in your templates.
* Hit **COPY TO CLIPBOARD** to grab the raw code.



---

## 🛠 Technical Details

* **Persistence:** Uses `localStorage` to keep your presets safe even after closing the tab.
* **Styling:** Custom CSS implementing a pixelated GUI system with CSS animations for the `&k` magic effect.
* **Zero Dependencies:** No frameworks or external libraries required (uses Google Fonts for the Minecraftia typeface).
